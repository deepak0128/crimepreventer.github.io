How to Use:

First of all Register A Suspect with the "Register Suspect" Button in the above section. The registration of a suspect will require name and photo(s), input both of them and then the suspect will be registered succssfully. So after registring suspect successfully, now we can use these data to prevent the crime. Now click on "Catch Suspect" button it will pop up a new interface and then select any video stream which is meant to be monitored and follow the simple instruction shown in those options. After selecting preferable stream now to start play the video, the monitoring will be started automatically and whenever a suspect face is match, program will alert us thus preventing the crime.
